# Elderly Mode

这个文件用于适老化设计和页面自适应规则。

## Scope

当任务涉及这些关键词时读取本文件：

- 适老化
- 字号放大
- 热区
- 老年模式
- 大字版
- 页面可读性

## Working Rules

- 适老化不是简单放大字体，要同时看布局、组件、热区、反馈和流程可理解性
- 数值阈值、字号变化映射、热区标准以官方页为准，不要凭经验写死
- 页面适配优先保证主流程可用，再做视觉微调

## What To Check

- 字体大小变化规则
- 界面自适应基础规则
- 基础组件适配规则
- 热区规则
- 页面适配示例
- 是否可以使用自动适配工具
