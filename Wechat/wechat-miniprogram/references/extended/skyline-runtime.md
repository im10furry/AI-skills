# Skyline Runtime

这个文件用于 Skyline 运行时能力、差异和迁移风险。

## Use This File When

- 用户明确提到 `Skyline`
- 任务涉及高性能渲染或特定渲染差异
- 需要判断 Skyline 和 WebView 渲染差异
- 需要做 Skyline 迁移或发布验证

## Working Rules

- 不要默认当前项目已经适配 Skyline
- 先确认组件能力、路由行为、调试方式和发布要求
- 遇到兼容性问题，优先读官方对比和迁移文档

## What To Check

- Skyline 的能力定位
- 组件差异
- 调试方式
- 兼容性与迁移限制
- 发布注意点
