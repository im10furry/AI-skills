# Security Domain And Gateway

这个文件用于域名约束、安全基线、内容安全和微信网关。

## Scope

- 业务域名
- HTTPS 约束
- HttpDNS
- 开发安全
- 风险能力
- 内容安全
- 微信网关
- 平台安全通知

## Working Rules

- 网络问题先查域名和 HTTPS，再查代码
- 安全问题先分清开发安全、内容安全、运行风险还是网关能力
- 涉及风控或安全通知时，不要凭经验给结论，直接回官方页
