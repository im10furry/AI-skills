# Plugins Functional Pages And Server Side

这个文件用于插件、功能页和服务端能力。

## Plugin Scope

- 插件开发
- 插件接入
- 插件 API 限制
- 插件组件限制

## Functional Page Scope

- 地址选择
- 发票
- 发票抬头
- 支付请求
- 用户信息功能页

## Server-Side Scope

- 后端 API
- 消息推送
- 服务端回调或服务端能力集成

## Working Rules

- 插件和普通小程序能力不同，先看限制页
- 功能页优先按官方指定接入方式，不要自己模拟同等能力
- 服务端能力不是前端 `wx.*` API，先区分调用侧
