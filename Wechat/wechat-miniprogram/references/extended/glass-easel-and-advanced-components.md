# Glass Easel And Advanced Components

这个文件用于 Glass-Easel 和高级组件系统能力。

## Use This File When

- 用户明确提到 `glass-easel`
- 需要链式 API、动态 slot、数据函数等高级组件能力
- 需要判断 Glass-Easel 与 Skyline 的关系
- 需要做高级组件系统迁移

## Working Rules

- 不要把 Glass-Easel 当成普通组件语法糖
- 先确认当前工程是否真的使用这套能力
- 迁移时优先看官方迁移页，不要直接套普通组件经验
