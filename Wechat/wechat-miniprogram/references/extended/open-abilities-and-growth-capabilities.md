# Open Abilities And Growth Capabilities

这个文件用于开放能力和增长能力索引。

## Scope

- 登录
- 授权
- 手机号
- 分享
- 订阅消息
- Launch App
- URL Scheme / URL Link
- 数据分析

## Working Rules

- 先确认目标是能力选型还是具体 API 实现
- 涉及账号、手机号、订阅消息、分享时，优先回开放能力专题页
- 增长和拉起类能力通常伴随平台规则，不要只写代码示例
