# Testing Automation And Quality

用于处理自动化测试、MiniTest、Automator、质量面板和体验巡检。

## Core Points

- 自动化和质量平台属于扩展开发集，不是所有项目默认都要接入。
- 先区分测试需求：
  - UI 自动化
  - 回放或录制
  - 云测与 MiniTest
  - 体验评分和质量巡检
- DevTools Automator 适合自动化操作和脚本化调试。
- MiniTest 适合质量测试、云真机或自动化验证场景。
- 体验评分和性能面板更适合发现问题，而不是替代功能测试。

## Typical Routing

- “如何脚本化操作开发者工具/小程序页面”：
  - 先看 Automator
- “如何做质量平台测试或云测”：
  - 先看 MiniTest
- “想拿一套质量结论”：
  - 先看体验评分、质量面板和性能诊断
