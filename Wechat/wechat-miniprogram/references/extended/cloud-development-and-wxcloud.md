# Cloud Development And Wxcloud

这个文件用于微信云开发的能力边界和官方入口导航。

## Scope

- 云开发是否适合当前项目
- 环境与资源共享
- 云函数
- 数据库
- 存储
- HTTP API
- 控制台与权限

## Working Rules

- 先确认项目是不是原生云开发架构，不要默认所有小程序都用云开发
- 云开发问题先区分控制台、环境、函数、数据库、存储哪一层
- 涉及权限和资源共享时，优先核对环境和账号模型
- 具体 API 设计、数据模型和安全规则问题要继续下钻官方专题页
