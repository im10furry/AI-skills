# CLA 初稿模板

以下内容用于快速生成可编辑版本。根据任务选择个人或企业版本，并保留占位。

## 1. 当事方

本贡献者许可协议由 `[贡献者姓名或企业名称]` 与 `[项目维护主体全称]` 就 `[项目或仓库名称]` 签署。

## 2. 贡献定义

“贡献”指贡献者提交、上传、发送或以其他方式提供给项目的代码、文档、配置、素材或其他工作成果，范围为 `[待确认: 仓库 / 组织 / 历史提交 / 未来提交]`。

## 3. 权利安排

### 许可型版本

贡献者保留其贡献中的权利，但授予项目维护主体一项 `[待确认: 全球 / 非独占 / 不可撤销 / 可再许可]` 的许可，以复制、修改、分发、公开展示、公开传播并在 `[待确认: 现有及未来许可证模型]` 下再许可该等贡献。

### 版权让与型版本

贡献者同意将其对贡献享有的 `[待确认: 著作权及相关权利]` 在适用法律允许范围内转让给项目维护主体；如无法完成转让，则授予前述最广许可作为替代。

## 4. 专利与其他权利

`[待确认: 是否包含专利授权、专利终止、精神权利同意或放弃、商标保留]`

## 5. 陈述与保证

贡献者声明其有权提交贡献，并在其所知范围内，贡献不会违反第三方对其具有约束力的义务；`[待确认: 是否限制保证范围，是否排除“绝对不侵权”类表述]`
