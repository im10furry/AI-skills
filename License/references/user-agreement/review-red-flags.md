# 用户协议红线审阅清单

## 产品不一致

- 协议写有订阅、退款、封禁、删除内容，但产品页面没有对应流程
- 产品已有 AI、社区、API、团队空间等功能，协议完全没提

## 表述过宽

- “我们可自行决定任何事项且无需通知”
- “用户承担全部责任”但平台保留广泛控制权
- “永久不可撤销授权”超出托管服务必要范围

## 缺失基础条款

- 没有主体信息
- 没有通知方式
- 没有终止或暂停规则
- 没有责任限制和免责声明
- 没有更新条款生效机制

## 强法域敏感项

- 自动续费
- 消费者退费
- 未成年人
- 平台内容责任
- 仲裁、管辖、集体诉讼弃权

## 审阅输出建议

- 缺失项
- 不一致项
- 高风险项
- 建议补写顺序
