# 用户协议初稿模板

## 使用方式

- 先按产品类型删除不适用章节。
- 对法域、自动续费、退款、争议解决、未成年人等敏感项保留占位，不要擅自替用户决定。
- 如果产品有 AI、社区、订阅、API 或平台双边交易，按需补对应条款块。
